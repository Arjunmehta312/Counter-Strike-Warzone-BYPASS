🎮 Counter-Strike WaRzOnE: Play with Friends on College Wi-Fi

This guide helps you set up and play CS WaRzOnE with friends using a local server.

⬇ Download Counter-Strike WaRzOnE.zip

Share this link with friends on the same Wi-Fi.

📌 Step 1: Share the Game Files

Run a local file-sharing server to distribute game files quickly.

1️⃣ Open the folder where "Counter-Strike WaRzOnE.zip" is stored.
2️⃣ Open PowerShell or Command Prompt and type:
   python -m http.server 8000 --bind 0.0.0.0
3️⃣ Find your IP address:
   ipconfig | findstr "IPv4"
4️⃣ Share the link with friends: 
   http://YOUR_LOCAL_IP:8000

📌 Step 2: Install & Start the Game

  • Download and extract the ZIP file.
  • Run CS16Launcher.exe to start the game.
  • Go to Options > Multiplayer > Customize and change your name.

📌 Step 3: Host a Server with HLDS

1️⃣ Open the extracted folder and run hlds.exe.
2️⃣ Set up your game preferences (map, settings, players, etc.).
3️⃣ Click "Start Server" and note your local IP.

📌 Step 4: Join the Game

  • Open **CS16Launcher.exe**.
  • Go to Find Servers > Favorites > Add Server.
  • Enter the server IP you got from HLDS.
  • If the server doesn’t show, open the console and type: connect YOUR_SERVER_IP:27015

📌 Step 5: Troubleshooting

❌ Game doesn’t appear in LAN? Try using the console command:

connect YOUR_SERVER_IP:27015

❌ Connection Timed Out? Make sure both players are on the same Wi-Fi network.

❌ HLDS Server Not Visible? Disable Windows Firewall temporarily:

netsh advfirewall set allprofiles state off

Enable it back after playing:

netsh advfirewall set allprofiles state on
